About This Project

Hi there – welcome to my project!

I’m from Melbourne, Australia, and like many of us, I’ve heard all the jokes about our “four seasons in one day” weather. So for my CS50 final project, I thought it’d be fun to build something that lets people compare weather across cities—My beautiful one included.

It is a light-hearted way to see real weather data and maybe challenge a few assumptions along the way.

And if you’re from Sydney... I suggest you to run a summer comparison and see who’s really getting drenched.

Built with:

  • Open-Meteo API – for historical weather and geolocation data
  • Python (Flask), HTML/CSS, and a dash of JavaScript
